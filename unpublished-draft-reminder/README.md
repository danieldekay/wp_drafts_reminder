Sends an email reminder to the admin if a draft has been unpublished for a certain period.

## Description

This plugin checks daily for drafts that have been unpublished for more than 7 days. If such drafts are found, it sends an email reminder to the admin.

## Installation

1. Upload the `unpublished-draft-reminder` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Frequently Asked Questions

### How do I change the reminder period?

Currently, the reminder period is fixed at 7 days. Future versions may include an option to customize this.
